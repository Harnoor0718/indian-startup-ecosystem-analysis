# Indian Startup Ecosystem Analysis – Funding & Unicorn Trends

Analyzed India's startup ecosystem (2015–2024) covering funding trends, sector growth, investor activity, geographic distribution and unicorn companies.

## Highlights
- Cleaned and standardized funding data (currency normalization, city name unification, deduplication) into a master dataset.
- Built EDA visualizations and an interactive dashboard to surface year-over-year funding growth, top sectors, and time-to-unicorn trends.
- Tech Stack: Python, Pandas, NumPy, Matplotlib, Seaborn, SQLite, Power BI

## Project Structure
data/raw/        - original downloaded datasets
data/cleaned/    - cleaned, processed datasets
notebooks/       - Jupyter notebooks for cleaning, EDA, analysis
reports/         - final report and exported notebook PDF
dashboard/       - dashboard screenshots and Power BI/Tableau link

## Research Questions
- Which sectors raised the most funding (2015–2024)?
- How has total startup funding grown year-over-year?
- Which cities and states dominate the startup landscape?
- How many unicorns has India produced, and which sectors produce them fastest?
- Which investors have the largest portfolio in the ecosystem?
